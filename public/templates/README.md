# Plantilla de Arqueo

Coloque aquí el archivo **arqueo-plantilla.xlsx** con su formato, colores y estructura.

Al exportar el arqueo desde el Historial de Ventas, el sistema rellenará automáticamente las filas de datos (desde la fila 16) preservando todo el formato original.

**Columnas que se rellenan:**
- Item, Descripción, Correlativo, Total Ingreso $
- Cash Dólares Ingreso, Zelle, PDV Shanklish $, PDV Superferro $, Pago Móvil $, Servicio 10%

Puede actualizar la plantilla en cualquier momento reemplazando el archivo.
